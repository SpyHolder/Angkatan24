## Instalasi
Pastikan menggunakan php versi 8.^

```bash
git clone https://github.com/SpyHolder/Angkatan24.git
cd Angkatan24
composer install
composer update
cp .env.example .env
php artisan key:generate
php artisan migrate
npm install
php artisan serve
```
